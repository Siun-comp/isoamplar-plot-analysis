IsoAmplar Plot Analysis - Icon Options

SVG candidates for favicon, browser tab, bookmark, and future PWA icons.

Option A - Curve Focus

Best when the chart itself should be the main signal.

Cleanest favicon candidate. It keeps the amplification curve and threshold readable at small sizes.

Option B - IA Mark

Best when the product name should feel more brand-like.

More logo-oriented. It reads well in a taskbar, but the curve is secondary to the IA identity.

Option C - Cartridge Motif

Best when the 1-sample / 8-reagent workflow should be visible.

Most domain-specific. It is useful for the app identity, but denser than A at 16px.
